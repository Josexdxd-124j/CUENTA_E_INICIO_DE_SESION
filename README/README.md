-CARACTERISTICAS:

Registro de Usuarios: Almacena los datos del usuario (Nombre, Correo, Celular, Contraseña) en el localStorage.

Validación de Formularios: Uso de expresiones regulares (RegEx) para asegurar la validez y seguridad de los datos ingresados.

Inicio de Sesión: Verifica las credenciales contra el usuario almacenado.

Sistema de Bloqueo: Bloquea la cuenta después de 3 intentos fallidos de inicio de sesión.

Recuperación de Contraseña: Permite actualizar la contraseña almacenada.

Funcionalidad 'Mostrar/Ocultar' Contraseña: Icono de "ojito" para alternar la visibilidad de la clave.

Diseño Responsivo: Estilos básicos y modernos con un degradado en el fondo.


Este proyecto incluye:
- Registro con validaciones por expresiones regulares
- Inicio de sesión
- Bloqueo por 3 intentos
- Recuperación de contraseña
- Mostrar/ocultar contraseña
- Almacenamiento en localStorage
- Publicado en GitHub Pages
